# Imersão Fullcycle 21 - Home Broker

## Descrição

Repositório do Next.js (front-end do home broker)

## Requerimentos

- Node.js
- Docker

## Rodar a aplicação

### Instalar as dependências

```bash
npm install
```

### Rodar o front-end

```bash
npm run dev
```

É necessário ter o Nest.js funcionando antes para rodar o front-end.
