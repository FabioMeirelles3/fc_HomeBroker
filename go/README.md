# Imersão Fullcycle 21 - Home Broker

## Descrição

Repositório do Golang (simulador de bolsa de valores).

## Requerimentos

- Docker instalado

## Rodar a aplicação

### Subir o Kafka

Execute o comando abaixo:

```bash
docker compose up
```

### Rodar o Simulador de Bolsa de Valores

Execute o comando abaixo:

```bash
go run cmd/trade/main.go
```
